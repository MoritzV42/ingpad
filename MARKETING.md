## Techniker-Forum / Fachschule (Deutsch) — primäre Zielgruppe

**Titel:** ingpad — technische Lernsituationen auf einer Canvas lösen (Stift + Rechnen + KI-Tutor)

**Body:**

```
Ich hab ein kleines Open-Source-Tool gebaut, das eine technische Lernsituation
(Maschinenbau / Mechatronik / Technikerschule) in eine interaktive Arbeitsflaeche
verwandelt:

- Rahmendaten/Vorgaben dauerhaft links sichtbar
- Aufgabe in Schritte zerlegt, jeder Schritt mit Gegeben / Gesucht / Ansatz
- pro Schritt ein Zeichenfeld zum handschriftlichen Rechnen (Stift, Zoom/Pan,
  hoehenverstellbar) - die KI prueft deinen Schritt, uebernimmt das Ergebnis und
  oeffnet den naechsten
- pro Aufgabe sind die Unterlagen UND die passende Seite der Formelsammlung verlinkt

Statt PDF + Taschenrechner + Zettel + PowerPoint getrennt: alles auf einer Flaeche.
Sokratisch - die KI rechnet nicht einfach vor, sondern fragt und prueft.

Open Source (MIT), laeuft lokal. Aufgaben-PDFs sind aus Copyright-Gruenden nicht
enthalten - du bringst deine eigenen mit.

Demo (Becherwerk-Antriebsstrang): [Loom]
Code: https://github.com/MoritzV42/ingpad

Feedback von anderen Technikerschuelern willkommen - was wuerdet ihr brauchen?
```

## r/EngineeringStudents (English)

**Title:** I built an open-source canvas to work through engineering exercises step by step (pen + AI tutor)

**Body:**

```
Working an exercise meant juggling a PDF, a calculator, paper and PowerPoint. So I
built ingpad: one canvas with the framework data pinned on the left, the task split
into steps (Given / Sought / Approach each), and a per-step drawing field where you
handwrite the calculation with a stylus. The AI checks the step, records the result
and opens the next one. Formula-sheet pages are linked per topic.

It's socratic on purpose - it asks and verifies instead of dumping the answer.

MIT, runs locally, no exercise PDFs bundled (bring your own).
Demo + repo: https://github.com/MoritzV42/ingpad
```
